Introduction

Watch this official presentation to understand IoT:

AWS IoT is a managed cloud platform that lets connected devices easily and securely interact with cloud applications and other devices. AWS IoT platform is an easy way to connect IoT-enabled devices to the cloud, store, analyze and visualize data. The cloud connection to IoT devices is fast and lightweight, which makes AWS IoT a great fit for devices that have limited processing power, battery life or memory. With AWS IoT platform we can create applications that enable our users to control these devices from their phones or tablets.

Various components can be used to connect IoT. An application accesses the service through AWS SDKs. AWS IoT APIs are divided into the control plane, which includes service configuration, device registration and logging; and the data plane, which includes data ingestion.

AWS IoT makes it easy to use AWS services like AWS Lambda, Amazon Kinesis, Amazon S3, Amazon Machine Learning, Amazon Dynamo DB, Amazon CloudWatch, AWS CloudTrail, and Amazon Elasticsearch Service with built-in Kibana integration, to build IoT applications that gather, process, analyze and act on data generated by connected devices, without having to manage any infrastructure.
